# zg-go.pl

Strona klubu Go **Semedori** w Zielonej Górze.
Hostowana na GitHub Pages → <https://zg-go.pl/>

Plik źródłowy: `index.html` (czysty HTML + CSS, bez frameworka, bez build-stepa).
Deploy: `git push` → GitHub Pages auto-deploy.
